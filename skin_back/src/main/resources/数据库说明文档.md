# 数据库设计文档

## 数据库概述
该数据库使用 MySQL 8.0.16 版本，字符集为 utf8mb4。数据库名称为 big_event，主要用于宠物商店管理系统。

## 表关系说明

1. 用户认证关系：
   - signon表通过userid关联account表

2. 商品关系：
   - item表通过productid关联product表，并且启用级联删除
   - product表通过category关联category表，并且启用级联删除
   - item表通过itemid关联inventory表

3. 订单关系：
   - orders表通过userid关联account表
   - lineitem表通过orderid关联orders表
   - lineitem表通过itemid关联item表
   - orderstatus表通过orderid关联orders表

4. 购物车关系：
   - cart表通过username关联signon表
   - cart表通过itemid关联item表

## 索引说明

1. 主键索引：
   - 所有表的主键字段都建立了主键索引

2. 外键索引：
   - item表：productid, supplier
   - product表：category
   - orders表：userid
   - lineitem表：orderid, itemid
   - orderstatus表：orderid
   - cart表：itemid, username

3. 普通索引：
   - product表：name
   - orders表：userid

## 注意事项

1. 所有表使用utf8mb4字符集，支持完整的Unicode字符
2. 订单状态使用varchar(2)类型，0表示未发货，1表示已发货
3. 价格相关字段使用decimal(10,2)类型，确保精确计算
4. 时间相关字段使用date类型
5. 所有表都使用InnoDB引擎，支持事务和外键约束
